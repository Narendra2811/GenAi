HUMAN-IN-THE-LOOP (LangGraph) — QUICK REVISION

1. interrupt()

* pauses graph execution
* returns "**interrupt**" in response
* can send structured data (dict)

2. Resume

* use Command(resume=value)
* value is returned inside interrupt()
* execution continues from same line

3. Flow
   Graph → interrupt → stop
   User input → Command(resume=...) → continue

4. Key Requirements

* checkpointer (mandatory)
* thread_id (mandatory)

5. Important Concepts

* resume value = interrupt return value
* execution resumes exactly where paused
* supports multiple interrupts

6. Use Cases

* user approval
* confirmation steps
* human correction
* tool validation

7. Common Errors

* no checkpointer → resume fails
* wrong thread_id → wrong state
* using llm() instead of invoke()

Memory Trick:
interrupt = pause + wait for human
resume = continue with human input
